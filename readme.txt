== Description ==

This plugin give writers an option to automatically add spaces before paragraphs.
It's mainly designed for Simplified Chinese users, because most of their paragraphs are indented by 4 spaces.
In edit page, if paragraphs are not start with 4 spaces, the checkbox for auto adding spaces will show up.
在文章编辑页，当有段落段首不以4个空格开头时，侧栏会出现“自动补足段首空格”的选项。

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
